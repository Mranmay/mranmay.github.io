# Mranmay Shetty

Personal academic/research website for Mranmay Shetty.

Built with Astro, Tailwind CSS, and Astro content collections. The site is
designed for GitHub Pages at:

```txt
https://mranmay.github.io
```

## Local development

```bash
npm install
npm run dev
```

## Blog posts

Blog posts live in `src/content/blog/` and can be Markdown or MDX.

Images exported from Google Docs can be placed under `public/images/posts/` and
referenced from Markdown as:

```markdown
![Figure description](/images/posts/figure-name.png)
```

## Build

```bash
npm run build
```
